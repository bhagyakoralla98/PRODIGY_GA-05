# PRODIGY_GA_05

## Task 05 - Neural Style Transfer

This project demonstrates Neural Style Transfer, a Generative AI technique that combines the content of one image with the artistic style of another image.

In this task, a content image and a style image were used to generate a new styled output image. The resulting image preserves the original content while applying the visual appearance of the style image.

### Files Included
- Content Image
- Style Image
- Output Image
- Colab Notebook

### Outcome
Successfully generated a stylized image by applying the artistic style of one image to another.
